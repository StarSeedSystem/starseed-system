# Agent Profile: Anchor

**Role**: System Dock & Core Access
**Position**: Bottom Border (Nadir)
**Color Identity**: Neutral Gray / Prism White / Red (Root)
**Philosophy**: Stability, Gravity, Root Access.

## Capabilities
- Main application navigation.
- Access to global tools (AI, Control Panel, Menu).
- "Red Mode" for deep navigation/root access.

## Interaction Model
- **Activation**: Mouse to bottom edge or swipe-up.
- **Behavior**: Spring-loaded dock emergence. Core nucleus of 3 buttons.
