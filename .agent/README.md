# AntiGravity Agent System

This directory houses the **Neural Layer** of the StarSeed Network. It contains the Skills, Agents, and Workflows that drive automation and intelligence.

## Components

### `skills/`
Modular capabilities that agents can utilize.
- Example: `ui-ux-pro-max` (Design Intelligence)

### `workflows/`
Standard Operating Procedures turned into automated sequences.

### `agents/`
Specialized AI Personas that operate the system.
- **Zenith**: Contextual Guide (Top)
- **Horizon**: Creation Canvas (Left)
- **Anchor**: System Dock (Bottom)
- **Logic**: Control Panel (Right)
- **Exocortex**: General System Pilot

### `mcp-servers/`
Configuration and logs for Model Context Protocol interactions.

### `memory/`
Short-term memory storage for active agent sessions.
